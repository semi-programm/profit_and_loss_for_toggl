# How to use
## Init(local)
- do composer update.
- create .env and setting.
>$ cp .env-local .env  
>$ php artisan key:generate  
>$ php artisan config:clear

## What command
- *all data* insert and update from toggl
>$ toggl:all
- workspaces data insert and update from toggl
